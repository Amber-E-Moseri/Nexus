# Regional Secretary User Guide
## BLW Canada Ministry Calendar System

**Role**: View-only access to all calendar spaces  
**What You Can Do**: View events, subscribe to feeds, export reports

---

## Your Permissions

- ✅ View Programs space events (read-only)
- ✅ View Admin space events (read-only)
- ✅ Subscribe to iCal feeds
- ✅ View approval queue (overview only)
- ✅ View sync status
- ❌ Cannot create events
- ❌ Cannot approve/reject events
- ❌ Cannot edit events
- ❌ Cannot delete events
- ❌ Cannot connect Google Calendar

---

## Viewing the Calendar

### Access the Calendar

1. **Go to Calendar Section**
   - Main navigation → "Calendar"
   - See both Programs and Admin spaces

2. **Choose Your View**
   - **Month View** (default): See full month at a glance
   - **List View**: Chronological event list with filters
   - **Week View**: Detailed week view

### Understanding Event Status

**Approved** ✓
- Fully confirmed and visible to all
- Ready for team members
- Synced to Google Calendar (if connected)
- **Color**: Green badge

**Pending** ⏳
- Awaiting manager approval
- Not yet fully visible
- Shows to limited people
- **Color**: Yellow badge

**Rejected** ✗
- Manager rejected this event
- Won't happen as planned
- May see reason why
- **Color**: Red badge

## Using iCal Subscriptions

### What Are Subscriptions?

iCal feeds let you add ministry calendar to personal calendar apps:
- Google Calendar
- Apple Calendar
- Outlook
- Any app supporting iCal format

**Benefit**: Automatic updates every 15 minutes without opening Nexus.

### Subscribe to a Feed

1. **Find Available Subscriptions**
   - Calendar section → "Available Subscriptions"
   - See list of public feeds

2. **Copy the Feed URL**
   - Click "Copy" button
   - Copies iCal URL to clipboard

3. **Add to Your Calendar**
   
   **For Google Calendar**:
   - Open Google Calendar
   - Click "+" next to "Other calendars"
   - Select "Subscribe to calendar"
   - Paste the URL
   - Click "Subscribe"
   
   **For Apple Calendar**:
   - Apple Calendar app
   - File → New Calendar Subscription
   - Paste the URL
   - Choose settings
   - Click "Subscribe"
   
   **For Outlook**:
   - Outlook app
   - Calendar → "Add calendar"
   - Select "Subscribe from web"
   - Paste the URL
   - Confirm

4. **Done!**
   - Calendar appears in your app
   - Updates automatically every 15 minutes
   - No need to open Nexus again

### Managing Subscriptions

**Update Frequency**:
- Calendar apps check for updates every 15 minutes
- Can manually refresh in most apps
- Changes visible within 30 minutes typically

**Remove Subscription**:
- Go to your calendar app
- Right-click the subscribed calendar
- Delete or unsubscribe
- Link still works for others

## Monitoring Google Calendar Sync

### View Sync Status

1. **Check Sync Info**
   - Calendar settings → "Google Calendar Sync"
   - See if Programs and Admin spaces are synced

2. **What to Look For**:

## FAQ

**Q: Why can't I edit events?**  
A: As Regional Secretary, you have read-only access for oversight. Contact Programs Manager to request changes.

**Q: Can I see who's attending events?**  
A: Not in the current system. Event details show who created it, not attendees. Attendance tracking planned for future.

**Q: What if I find an error in an event?**  
A: Contact the Programs Manager who created it. They can edit or delete it.

**Q: Why didn't an event appear on my calendar?**  
A: Possible reasons:
1. Event still pending (not approved)
2. Calendar app hasn't synced (wait 15 min)
3. Subscription was public but later made private
4. URL was copied incorrectly

**Q: Can I share subscription links?**  
A: Only if they're marked "Public". Ask the Programs Manager if you need to share.

**Q: Do I need to be online to see updated events?**  
A: No. Once subscribed, calendar app updates automatically. Can view offline but updates won't appear until next sync.

**Q: What happens if the programs manager deletes a subscription?**  
A: Link stops working. Calendar no longer updates. You'll need to use a different feed or ask for a new one.
