# Admin Manager User Guide
## BLW Canada Ministry Calendar System

**Role**: Full control of Admin space calendar and events  
**What You Can Do**: Create, edit, delete, approve, and manage Admin space events and subscriptions

---

## Overview

This guide is **nearly identical to the Programs Manager Guide**, with one key difference:

- ✅ You manage **Admin space** events
- ❌ You cannot access **Programs space**
- Everything else is the same

For full details, refer to **PROGRAMS_MANAGER_GUIDE.md** and replace "Programs" with "Admin" in your mind.

---

## Quick Reference

### Your Permissions

- ✅ Create events in **Admin space** (not Programs)
- ✅ Edit events in Admin space
- ✅ Delete events in Admin space
- ✅ Approve/reject pending Admin events
- ✅ Connect Google Calendar to Admin space
- ✅ Create iCal subscriptions for Admin space
- ❌ Cannot access Programs space events
- ❌ Cannot manage other spaces

---

## Key Differences from Programs Manager

| Task | Programs Manager | Admin Manager |
|------|-----------------|--------------|
| Create events | Programs space | Admin space |
| Approve events | Programs only | Admin only |
| Google Calendar | Connects Programs Cal | Connects Admin Cal |

## Core Workflow (Same as Programs Manager)

### 1. Create Event
- Admin space → Calendar → "Create Event"
- Fill form (title, dates, etc.)
- Submit → event starts as "Pending"

### 2. Approve Events
- Dashboard → "Event Approvals"
- Review pending events
- Approve or reject
- Add notes for rejection

### 3. Connect Google Calendar
- Admin space → Settings → "Calendar Integrations"
- Click "Connect Google Calendar"
- Approve Google permissions
- Automatic sync every 15 minutes

### 4. Create Subscriptions
- Admin space → "Subscriptions" → "Create"
- Set filters and name
- Copy URL and share
- People can add to their calendar apps

### 5. View Events
- Calendar view (month, week, list)
- Filter by status, priority, type
- Click to see details
- Edit or approve as needed

## Important Notes

### Space Isolation
- **Admin events stay in Admin space**
- Programs Manager cannot see them (and you can't see Programs events)
- This is intentional for security

### Google Calendar
- **Separate calendar**: Your Google Calendar connection is independent
- **Only Admin events sync** to your connected Google Calendar
- Programs Manager connects their own Google Calendar for Programs events

### Subscriptions
- **Admin-only subscriptions**: Can only include Admin space events
- Share links only with people who should see Admin space
- Can filter by priority/status like Programs Manager can

## FAQ (Admin Manager Specific)

**Q: Can I see Programs space events?**  
A: No. Space isolation is by design. Ask Programs Manager if you need event details.

**Q: Why is my Google Calendar empty?**  
A: Events only sync if they're in Admin space AND approved. Check event status.

**Q: Can Programs Manager see my Admin events?**  
A: No. Each manager only sees their own space.

**Q: What if I need to see both spaces?**  
A: Ask Regional Secretary — they have read-only access to both.

**Q: Can I subscribe to Programs events?**  
A: Only if Programs Manager creates a public subscription. You can't create one for Programs space.

## Detailed Instructions

### Creating an Event
1. Navigate to Admin space
2. Click Calendar → Create Event
3. Fill in event details (same as Programs Manager guide)
4. Select "Create Event"
5. Event appears pending in your approval queue

### Approving Events
1. Go to Dashboard → Event Approvals
2. See all pending events (yours and other Admin Managers')
3. Click event to review
4. Click "✓ Approve" to confirm
5. Event immediately visible and syncs to Google

### Managing Google Calendar
1. Go to Admin space settings
2. Find "Google Calendar Integrations" section
3. Click "Connect Google Calendar"
4. Complete Google login (same space as Programs Manager)
5. After connecting:
   - Click "Sync Now" for immediate sync
   - System auto-syncs every 15 minutes
   - See "Last synced" timestamp

### Creating Subscriptions
1. In Admin space, find "Subscriptions" section
2. Click "Create Subscription"
3. Enter details:
   - Name: "Admin Calendar 2026"
   - Optional: Description, priority filter, status filter
   - Check "Public" to allow sharing
4. System generates URL
5. Click "Copy" to copy to clipboard
6. Share via email, Slack, etc.

## Troubleshooting

### Events Not Syncing to Google
**Check**:
1. Event status is "Approved" (not pending)
2. Google Calendar is connected (shows ✓ Connected)
3. Wait 15 minutes for automatic sync
4. Click "Sync Now" for instant sync

### Subscription Link Not Working
**Check**:
1. Subscription is set to "Public"
2. Link was copied correctly
3. Calendar app supports iCal (Google, Apple, Outlook do)
4. Try copying and pasting again

### Can't Approve Events
**Check**:
1. Event hasn't already been approved
2. You're in Admin space (not Programs)
3. You have Admin Manager role
4. Event belongs to Admin space

### Google Calendar Won't Connect
**Check**:
1. Using correct Google account
2. Not already connected to Programs space
3. Granting calendar permissions
4. Allowing offline access

## Best Practices

### Naming
- Clear, specific titles: "Admin Board Meeting Q2"
- Include location if relevant: "Board Room" or "Virtual"
- Add context: "Board Approval Needed" instead of just "Meeting"

### Scheduling
- Avoid conflicts with major Programs events
- Consider staff availability
- Add buffer time between events
- Flag conflicting events

### Approvals
- Approve events within 24 hours
- Add notes when rejecting
- If unsure, discuss with Programs Manager (if related)
- Keep approval queue clean

### Subscriptions
- Use meaningful names
- Filter appropriately (don't overwhelm with everything)
- Update descriptions if filters change
- Monitor access counts
